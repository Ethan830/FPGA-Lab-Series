# FPGA-Lab-Series
Repo for Field-Programmable Gate Array (FPGA) experiments that explores FPGA electronics, logics, and functionalities. Experiments include:
1. Logic Gates & Binary Arithmetic
2. Arithmetic Logics
3. Prime Number Detector & Follow the Yellow Brick Road
4. 4-bit Enabled Registed & JK Flip-Flop
5. Sequence Detector
6. Structural Modeling for a Ripple Carry Adder
7. Microprocessor Components: Sign Extension, ALU
8. Memory
9. MIPS Assembly and Machine Code
10. MIPS Microarchitecture
11. UART Serial Commnuication
